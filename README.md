# ESP Home Configs

Here are a bunch of configs I use (and have used) on various boards, in various configurations/usages, around my house.

## Boards

### In Use

* ABRobot (Teyleten) SuperMini ESP32-C3 OLED
   * BME280 over I2C
   * Home Assistant BLE Proxy
   * shows its own climate measurements on the OLED display
   * two matching configs spread around the house
   * one other config for testing on a breadboard
* Waveshare ESP32-C6 GEEK
  * BME280 over I2C
  * Home Assistant BLE Proxy
  * shows info from around the house on the TFT display
 
### Others

* Waveshare ESP32-S3 1.54inch ePaper
  * BME280 over I2C
  * Home Assistant BLE Proxy
  * shows it's own climate info on the ePaper display
  * replaced by SuperMini and Geek boards so the ePaper and S3 can be used for something else
* Adafruit QT Py ESP32-S3
  * BME280 over I2C
  * Home Assistant BLE Proxy
  * replaced by SuperMini and Geek boards so the more powerful S3 can be used for something else
